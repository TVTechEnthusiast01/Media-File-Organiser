# MediaOrganizer
This is a simple Media Organizer tool for downloading and uploading files on the server by categories. The file manager is made up with Node.js and Express. 

To begin application follow the following steps. 

1. Download and unzip the folder and place the folder within the OS(C:)/Users/("System User" e.g LVC)

2. Run git bash

3. Type the following commands to install and run the application. 

- cd MediaOrganizer 
- npm install 
- node app.js 

4. Redirect to the following url "http://localhost:9000/"
